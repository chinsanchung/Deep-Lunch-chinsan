# 0강
- ML과 DL의 차이점
  - ML은 사람의 가이드가 필요함. DL 은 가이드가 필요하지 않음.
  - DL 은 알아서 문제를 인식하고 답을 구할 수 있게 됨. ML 보다 DL 에서 풀 수 있는 문제가 더 많음
- DL 이란
  - DL 은 신경망으로 ML 을 수행함
  - 이미지 분류, 물체 감지, 순서가 있는 데이터를 분석하고 예측하는데 유용함.

# 1강 ML 개념

- 기계가 혼자 알고리즘을 작성해서 문제를 해결
- 종류
  - 지도학습: 정답이 있는 데이터로 예측 모델을 생성
  - 비지도학습: 정답이 없는 데이터로 분류 모델을 생성
  - 강화학습: 보상이 무엇인지를 알려줘서 스스로 보상이 높은 모델을 생성
- 예측을 위한 가중치를 계산하는 작업을 컴퓨터가 수행하도록 함
  - 데이터를 전달하여 가중치를 구하도록 명령. 
  - 컴퓨터가 구한 예측과 실제 데이터를 비교해서 오차가 최대한 적은 것을 선택.


# 2강 DL의 신경망

- 정교한 예측을 위해 perceptron 으로 중간 레이어를 추가한 것. 뇌의 작동 방식을 본따 신경망이라고 명칭.
- 레이어는 생각을 저장하는 공간
  - 초기 데이터에서 가중치를 둬서 계산해 중간 레이어에 저장하고, 그것으로 예측 모델을 생성.
  - 데이터의 특성을 컴퓨터가 알아서 도출해서 예측을 수행.
